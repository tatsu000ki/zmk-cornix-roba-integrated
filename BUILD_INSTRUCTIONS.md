# ビルド状況の確認とファームウェアのダウンロード

## 最新の変更内容

ZMK v0.3-branchとtemp-layer-thresholdモジュールを追加し、PMW3610ドライバーとの互換性を確保しました。

### 現在の設定

- **ZMKバージョン**: v0.3-branch（roBaの元の設定と同じ）
- **Cornixモジュール**: zmk-keyboard-cornix v2.6.8（外部モジュール）
- **PMW3610ドライバー**: kumamuk-git/zmk-pmw3610-driver
- **temp-layer-threshold**: yamaryu211/zmk-input-processor-temp-layer-threshold
- **zmk-helpers**: urob/zmk-helpers
- **ビルドワークフロー**: v0.2

## GitHub Actionsでビルド状況を確認

1. **GitHubリポジトリにアクセス**:
   ```
   https://github.com/tatsu000ki/zmk-cornix-roba-integrated
   ```

2. **「Actions」タブをクリック**

3. **左側の「Build ZMK firmware」ワークフローを選択**

4. **最新のワークフロー実行を確認**:
   - ✅ **緑のチェックマーク** = ビルド成功
   - ❌ **赤のバツマーク** = ビルド失敗
   - 🟡 **黄色の円** = ビルド実行中

## 手動でビルドを実行する方法

自動ビルドが開始されない場合：

1. GitHubリポジトリの「Actions」タブに移動
2. 左側の「Build ZMK firmware」を選択
3. 右側の「Run workflow」ボタンをクリック
4. 「Run workflow」を再度クリックして実行

## ビルド成功時の手順

### 1. ファームウェアのダウンロード

1. ワークフロー実行の詳細ページを開く
2. 下部の「Artifacts」セクションを確認
3. `firmware.zip`をダウンロード
4. ZIPファイルを解凍

### 2. ファームウェアファイルの確認

解凍すると以下のファイルが含まれています：

- **`cornix_left-zmk.uf2`** - Cornix左手用ファームウェア
- **`seeeduino_xiao_ble-roBa_R-zmk.uf2`** - roBa右手用ファームウェア

## ファームウェアの書き込み方法

### Cornix左手（cornix_left）の書き込み

1. Cornix左手のリセットボタンを**2回素早く**押す
2. USBストレージデバイスとして認識される（`NICENANO`または`CORNIX`という名前）
3. `cornix_left-zmk.uf2`をドラッグ&ドロップ
4. 自動的に再起動してファームウェアが適用される

### roBa右手（seeeduino_xiao_ble）の書き込み

1. Seeeduino XIAO BLEのリセットボタンを**2回素早く**押す
2. USBストレージデバイスとして認識される（`XIAO-SENSE`という名前）
3. `seeeduino_xiao_ble-roBa_R-zmk.uf2`をドラッグ&ドロップ
4. 自動的に再起動してファームウェアが適用される

## ペアリング方法

### 初回ペアリング

1. **roBa右手（セントラル/親機）を先に起動**
2. **Cornix左手（ペリフェラル/子機）を起動**
3. 自動的にペアリングが開始される
4. PCとのBluetoothペアリング：
   - PCのBluetooth設定を開く
   - 「Cornix」または「roBa」という名前のデバイスを探す
   - ペアリングを実行

### ペアリングのリセット

うまくペアリングできない場合：

1. PCのBluetooth設定から既存のペアリングを削除
2. 両方のキーボードの電源を切る
3. roBa右手を先に起動
4. Cornix左手を起動
5. 再度ペアリングを試す

## 期待される動作

### 左手（Cornix）
- **キー数**: 50キー
- **役割**: ペリフェラル（子機）
- **機能**: 通常のキー入力、レイヤー切り替え

### 右手（roBa）
- **キー数**: 41キー
- **トラックボール**: PMW3610センサー
- **役割**: セントラル（親機）
- **機能**: 
  - 通常のキー入力
  - マウス移動
  - マウスクリック
  - スクロール機能（レイヤー5）
  - レイヤー切り替え

### レイヤー同期
- 左右のキーボードのレイヤーは自動的に同期
- どちらの手でレイヤーを切り替えても、両方に反映される

## トラブルシューティング

### ビルドが失敗した場合

1. ワークフロー実行の詳細ページを開く
2. 失敗したジョブをクリック
3. エラーメッセージを確認
4. 以下の情報を報告：
   - エラーメッセージ
   - 失敗したステップ
   - ビルドログ

### ファームウェアは書き込めたが動作しない

1. **ファームウェアが正しく書き込まれているか確認**
   - 両方のキーボードに正しいファームウェアを書き込んだか
   - ファイル名を確認（左手用と右手用を間違えていないか）

2. **Bluetooth設定をリセット**
   - PCのBluetooth設定からデバイスを削除
   - 両方のキーボードを再起動
   - ペアリングをやり直す

3. **バッテリーを確認**
   - 両方のキーボードが充電されているか確認

4. **接続順序を確認**
   - roBa右手（親機）を先に起動
   - その後Cornix左手（子機）を起動

### トラックボールが動作しない

1. **roBa右手のファームウェアを確認**
   - `seeeduino_xiao_ble-roBa_R-zmk.uf2`が正しく書き込まれているか

2. **PMW3610センサーの接続を確認**
   - ハードウェアの配線が正しいか確認

3. **スクロールレイヤーを確認**
   - レイヤー5でスクロール機能が有効になる
   - 正しいレイヤーに切り替わっているか確認

## 次のステップ

ビルドが成功したら：

1. ✅ ファームウェアをダウンロード
2. ✅ 両方のキーボードにファームウェアを書き込む
3. ✅ ペアリングを実行
4. ✅ 動作確認
5. ✅ キーマップのカスタマイズ（必要に応じて）
